## 1. 📊 Opportunity Snapshot

| Field | Value |
|-------|-------|
| Customer | 华为云 · 电信/ICT · New Logo |
| Opportunity Name | 混合云迁移 — 核心数据中心退役 |
| Deal Value | $4.8M TCV |
| Current Stage | Technical Validation |
| Target Launch Date | 2026-Q4 |

### Why Now

华为云内部战略转型，董事会在2026年Q1财报会议上明确要求：2026年底前将核心数据中心运维成本降低35%。[网络搜索] 当前合同将于2026年12月31日到期，如果不在Q3前完成技术验证，将被迫续签现有高成本合约。

CTO 张伟在上次内部技术委员会上提出"云原生优先"战略，获得CEO支持。这意味着本次迁移已上升为公司战略级项目。[销售确认]

### Deal Objective

通过混合云架构帮助华为云在2026年Q4前完成核心工作负载迁移，实现年化运营成本节约$1.6M，同时确保99.99%可用性SLA满足其电信级客户要求。

### Win Strategy

核心差异化在于我们在电信行业的深度参考案例（中国移动、中国联通已成功迁移）以及AWS Outposts混合架构能力，完美契合华为云"本地+云端"的合规要求。客户最关注的是迁移过程中的零停机保障——我们的迁移加速器工具链（Migration Hub + Application Migration Service）在同类项目中有92%一次性成功率记录。

**Key Risks:** 竞争对手Azure已经在做POC，如果我们不能在6月底前展示优于Azure的TCO和迁移速度，可能失去技术团队支持。CFO对云支出有严格把控，需要强有力的财务论证才能通过预算审批。
